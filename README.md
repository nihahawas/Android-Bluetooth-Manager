# Android Bluetooth Manager 📱

A native Android application developed in Kotlin that provides a comprehensive interface for managing Bluetooth hardware.

## 🚀 Features
* **Power Management:** Toggle Bluetooth ON and OFF directly from the UI.
* **Discoverability:** Make the device visible to nearby hardware for a set duration.
* **Device Listing:** Retrieve and display a list of all currently paired (bonded) devices.
* **Dynamic UI:** Responsive design using `LinearLayout` with a custom color-coded button scheme.

## 🛠️ Technical Details
* **Language:** Kotlin
* **Minimum SDK:** API 23 (Android 6.0)
* **Key API:** `BluetoothAdapter`, `BluetoothDevice`
* **Permissions handled:** * `BLUETOOTH_CONNECT`
    * `BLUETOOTH_SCAN`
    * `ACCESS_FINE_LOCATION`
